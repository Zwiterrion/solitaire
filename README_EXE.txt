Pour choisir la version du projet que vous voulez lancer, vous pouvez taper les commandes suivantes:

make -- Lance la version finale du jeu

make VERSION=1 -- Lance une fenêtre contenant un bouton Quitter
make VERSION=2 -- Lance une fenêtre contenant le menu (non actif)
make VERSION=3 -- Lance une fenêtre contenant le score en haut
make VERSION=4 -- Lance une première version du jeu classique entièrement jouable
make VERSION=5 -- Jeu classique avec les coups joués affichés
make VERSION=6 -- Jeu complémentaire compris dedans
